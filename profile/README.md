# Welcome to TinyAlg

**TinyAlg** is a collection of open-source passion projects by lwgena at the intersection of tiny algorithms and digital well-being. Explore Chrome extensions, GitHub Actions, and technical insights for a healthier digital life.

## 🚀 Featured Product: Watchtime Badge
**Stop Binge-Watching Today.**
Visualize your actual time spent on YouTube, Netflix, and Prime Video directly on your browser icon.

- **Privacy First**: Designed with "No External Tracking". All viewing records are stored locally on your device.
- **Smart Tracking**: Uses *Attention-based Time Tracking* logic to measure true engagement, not just playback.
- **Lightweight**: No cloud uploads or background sync. Won't slow you down.

## 🛠 Technical Roots
This project originates from embedded systems design, creating resources that are approachable yet comprehensive:
- **Microcontroller Solutions**: Efficient algorithms for ESP32 and real-time systems.
- **Experimental Tools**: Projects like [`Waveu`](https://github.com/tinyalg/waveu) for designing waveforms.
